Vortex Virtual Environment (VVE)
O Vortex é uma solução de infraestrutura efêmera projetada para fornecer ambientes de desktop Linux via navegador em dispositivos móveis.

✨ Características:
Ambiente Efêmero: Destruição automática após o uso.

Sessão Otimizada: Ciclos de 30 minutos para conformidade com o GitHub.

Acesso Web: Sem necessidade de aplicativos externos (Powered by Cloudflare).

🚀 Como Iniciar:
Inicie um GitHub Codespace neste repositório.

No terminal, execute: bash setup.sh

Clique no link gerado pela Cloudflare para acessar sua área de trabalho.
